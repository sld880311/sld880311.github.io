vsftpd

  1. 查看是否安装：rpm –qa|grep vsftpd
  2. 安装：yum -y install vsftpd
  3. 启动：service vsftpd start
  4. 重启：service vsftpd restart
  5. 设置相关参数：setsebool -P ftp_home_dir 1
